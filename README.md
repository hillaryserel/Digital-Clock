# Digital Clock with Glowing Effect

This is a simple digital clock web application created using HTML, CSS, and JavaScript. The clock displays the current time and updates in real-time.

## Description

This web page features a digital clock with a glowing effect created using JavaScript. The clock displays the current time in hours, minutes, and seconds with an AM/PM indicator.

## Usage

To use this digital clock, simply open the `index.html` file in a web browser. The clock will automatically start and display the current time.

## Code Explanation

- The HTML file (`index.html`) contains the structure of the clock, including the necessary elements and a reference to an external stylesheet (`style.css`).
- The JavaScript code within the `<script>` tag fetches the current time and updates the clock every second using the `setInterval()` function.
- The `setInterval()` function continuously updates the displayed time by obtaining the current time from the `Date` object and formatting it.

## File Structure

- `index.html`: Contains the HTML structure of the digital clock.
- `style.css`: External CSS file for styling the clock (not provided in the code snippet).

## Getting Started

1. Download or clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
3. The digital clock will start displaying the current time.
